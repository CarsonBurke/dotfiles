# Review Own Changes

Inspect and test the task's final diff, then give a fresh read-only reviewer the request, repository instructions, and raw scope without leaking expected findings. Validate its concerns, fix material issues, rerun affected checks, and request focused follow-up when needed. Report accepted and declined findings, validation, and residual risk.

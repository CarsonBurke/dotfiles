# Analyze Stock

Analyze a stock ticker over a given timeframe (default: last month).

## Data gathering

1. Use ibkr MCP tools for news headlines, historical price bars, contract details, and current position.
2. If IBKR news is thin or stale, supplement with web search for recent news, analyst coverage, and controversy.

## Analysis

Cover: price action, sentiment (with evidence), controversy & risk, key events, bull/bear case, and current exposure if held.
